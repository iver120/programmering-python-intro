Intro til løkker: while og for

Lær å gjenta kode med while og for-løkker, inkludert break og continue.

← Til hovedsiden

Hva er en løkke?

En løkke gjør at kode kan kjøres mange ganger automatisk, i stedet for å skrive samme kode om og om igjen.

Python har to hovedtyper løkker:

  • while - kjører så lenge en betingelse er sann
  • for - kjører et bestemt antall ganger eller over elementer i en samling

while-løkker

En while-løkke sjekker en betingelse først, og kjører koden inne i løkka så lenge betingelsen er True.

# Grunnleggende while-løkke
teller = 1
while teller <= 3:
    print("Runde", teller)
    teller = teller + 1
print("Ferdig!")

# Utskrift:
# Runde 1
# Runde 2  
# Runde 3
# Ferdig!

# Løkke med input
svar = ""
while svar.lower() != "stopp":
    svar = input("Skriv 'stopp' for å avslutte: ")
    print("Du skrev:", svar)
print("Takk for nå!")
Viktig: Husk å endre variabelen som sjekkes, ellers får du en uendelig løkke!

for-løkker med range()

for-løkker brukes når du vet hvor mange ganger noe skal kjøres. range() lager en sekvens av tall[163][167].

# for med range() - kjør 5 ganger
for i in range(5):
    print("Gang nummer", i)

# Utskrift:
# Gang nummer 0
# Gang nummer 1
# Gang nummer 2
# Gang nummer 3
# Gang nummer 4

# range() med start og stopp
for tall in range(1, 4):
    print("Tall:", tall)

# Utskrift:
# Tall: 1
# Tall: 2
# Tall: 3
range(5) gir tallene 0, 1, 2, 3, 4. range(1, 4) gir 1, 2, 3 (stopp-tallet er ikke med).

for-løkker over tekst

for-løkker kan også gå gjennom hver bokstav i en tekst:

navn = "Anna"
for bokstav in navn:
    print("Bokstav:", bokstav)

# Utskrift:
# Bokstav: A
# Bokstav: n
# Bokstav: n
# Bokstav: a

break og continue

break stopper løkka helt, continue hopper til neste runde.

# break - stopp løkka
for i in range(10):
    if i == 3:
        break
    print("Tall:", i)

# Utskrift:
# Tall: 0
# Tall: 1
# Tall: 2

# continue - hopp over
for i in range(5):
    if i == 2:
        continue
    print("Tall:", i)

# Utskrift:
# Tall: 0
# Tall: 1
# Tall: 3
# Tall: 4

Oppgaver

  1. Lag en while-løkke som teller ned fra 5 til 1, og skriver "Start!" til slutt.
  2. Bruk for og range() til å skrive ut alle partall fra 0 til 10.
  3. Lag en while-løkke som ber brukeren gjette et hemmelig tall (du bestemmer), til de gjetter riktig.
  4. Bruk for til å gå gjennom navnet ditt og tell hvor mange vokaler det er.
