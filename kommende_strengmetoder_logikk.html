Kommende: Strengmetoder og logikk

Bygger videre på variabler, vilkår og datatyper med nyttige strengmetoder og logiske operatorer.

Mål for denne leksjonen

  • Lære sentrale strengmetoder: .strip(), .upper(), .lower(), .replace(), .startswith(), .endswith()
  • Forstå og bruke logiske operatorer: and, or, not
  • Kombinere strengmetoder og logikk i enkle if/elif/else‑programmer

Strengmetodene under er innebygde i str‑typen, og and/or/not er del av uttrykksspråket i Python.

Strengmetoder vi bruker nå

# .lower(): gjør alle bokstaver små
s = "HeI"
print(s.lower())  # "hei"

# .upper(): gjør alle bokstaver store
s = "Hei"
print(s.upper())  # "HEI"

# .strip(): fjerner mellomrom/tegn i starten og slutten
s = "   hei verden   "
print(s.strip())  # "hei verden"

t = ",,,hei..."
print(t.strip(",."))  # "hei"

# .replace(gammelt, nytt): bytter tekst
s = "hei verden"
print(s.replace("verden", "Python"))  # "hei Python"

# .startswith(prefix) og .endswith(suffix): sjekk start/slutt
fil = "bilde.png"
print(fil.endswith(".png"))    # True
print(fil.startswith("bil"))   # True
Disse metodene returnerer nye strenger og endrer ikke originalen, siden str er uforanderlig (immutable).

Logiske operatorer

Logiske operatorer kombinerer betingelser i if/elif/else og gir True eller False tilbake.

# and: begge må være sanne
alder = 16
poeng = 82
if alder < 18 and poeng >= 80:
    print("Ung og kvalifisert")

# or: minst én er sann
valg = "ja"
if valg == "ja" or valg == "j":
    print("Valgt JA")

# not: inverterer sannhetsverdi
regn = False
if not regn:
    print("Ta med solbriller")
Tips: Klammer kan brukes for å gjøre uttrykk tydeligere, f.eks. (x > 0 and y > 0) or z == 1.

Kombinasjon i praksis

Kombiner strengmetoder og logikk for mer robust input uten å bruke løkker eller exceptions.

# 1) Case-uavhengig ja/nei
svar = input("Vil du fortsette? (ja/nei): ").strip().lower()
if svar == "ja":
    print("OK, videre")
elif svar == "nei":
    print("Stopper")
else:
    print("Skriv 'ja' eller 'nei'")

# 2) Valider enkel filtype
fil = input("Filnavn: ").strip()
if fil.endswith(".png") or fil.endswith(".jpg"):
    print("Ser ut som et bilde")
else:
    print("Ukjent filtype")

# 3) Rydde og sjekke start/slutt
tekst = input("Skriv en hilsen: ").strip()
if tekst.lower().startswith("hei") and not tekst.endswith("!"):
    print("Fin hilsen, men legg gjerne til et utropstegn")

Øvingsoppgaver

  1. Be om et navn og skriv “Hei, Navn!” uansett hvordan brukeren skriver store/små bokstaver (bruk .strip() og .lower()/.upper()).
  2. Be om en e‑postadresse og sjekk veldig enkelt om den slutter på “.no” eller “.com”.
  3. Be om et filnavn og skriv “Video” hvis det slutter på “.mp4” eller “.mov”, ellers “Ikke video”.
  4. Lag en liten meny som aksepterer både “ja/j” og “nei/n” ved hjelp av .lower() og and/or.
  5. Be om en setning og bytt ut et ord med et annet ved hjelp av .replace().

Huskeliste

  • input() gir str, så strengmetoder brukes direkte på input‑verdier
  • .strip() rydder start/slutt, .lower()/.upper() normaliserer bokstaver
  • and krever begge sanne, or krever minst én sann, not snur sann/usann
  • Bruk flere små, tydelige vilkår fremfor ett langt uleselig uttrykk
