Lister og enkle operasjoner

Lær å lage og jobbe med lister: legge til, fjerne, og gå gjennom elementer.

Hva er en liste?

En liste er en samling av elementer som kan være tall, tekst eller andre typer data. Lister kan endres - du kan legge til og fjerne elementer.

# Lage lister
tall_liste = [1, 2, 3, 4, 5]
navn_liste = ["Anna", "Ola", "Kari"]
blandet = [1, "hei", 3.14, True]
tom_liste = []

print(tall_liste)    # [1, 2, 3, 4, 5]
print(navn_liste)    # ['Anna', 'Ola', 'Kari']

Hente elementer fra liste (indeksering)

Hvert element i en liste har en posisjon (indeks). Første element har indeks 0.

# Hente elementer
frukt = ["eple", "banan", "appelsin"]

print(frukt[0])    # "eple" (første)
print(frukt[1])    # "banan" (andre)  
print(frukt[2])    # "appelsin" (tredje)

# Negative indekser (baklengs)
print(frukt[-1])   # "appelsin" (siste)
print(frukt[-2])   # "banan" (nest siste)

# Endre elementer
frukt = ["eple", "banan", "appelsin"]
frukt[1] = "pære"
print(frukt)  # ["eple", "pære", "appelsin"]
Husk: Første element har indeks 0, ikke 1!

Lengde av liste: len()

len() forteller hvor mange elementer som er i lista.

tall = [10, 20, 30, 40]
print(len(tall))      # 4

navn = ["Per", "Ane"]
print(len(navn))      # 2

tom = []
print(len(tom))       # 0

Legge til elementer: .append()

.append() legger til et element på slutten av lista[165][169].

dyr = ["hund", "katt"]
print(dyr)           # ["hund", "katt"]

dyr.append("fugl")
print(dyr)           # ["hund", "katt", "fugl"]

dyr.append("fisk")
print(dyr)           # ["hund", "katt", "fugl", "fisk"]

Fjerne elementer

# .remove() - fjern første forekomst
farger = ["rød", "blå", "grønn", "blå"]
farger.remove("blå")
print(farger)  # ["rød", "grønn", "blå"]

# .pop() - fjern siste element
tall = [1, 2, 3, 4]
siste = tall.pop()
print(siste)     # 4
print(tall)      # [1, 2, 3]

# .pop(indeks) - fjern på bestemt plass
første = tall.pop(0)
print(første)    # 1  
print(tall)      # [2, 3]

for-løkker med lister

for-løkker er perfekte for å gå gjennom alle elementer i en liste[167]:

# Gå gjennom alle elementer
hobbyer = ["fotball", "lesing", "gaming"]

for hobby in hobbyer:
    print("Jeg liker", hobby)

# Utskrift:
# Jeg liker fotball
# Jeg liker lesing  
# Jeg liker gaming

# Med indeks og verdi
navn = ["Anna", "Ola", "Kari"]

for i in range(len(navn)):
    print(f"{i}: {navn[i]}")

# Utskrift:
# 0: Anna
# 1: Ola
# 2: Kari

Praktisk eksempel: Handleliste

print("=== Handleliste ===")
handleliste = []

# Legg til varer
handleliste.append("melk")
handleliste.append("brød") 
handleliste.append("epler")

print("Du må handle:")
for vare in handleliste:
    print("-", vare)

# Fjern en vare
handleliste.remove("brød")
print("\nOppdatert liste:")
for vare in handleliste:
    print("-", vare)

print(f"\nDu har {len(handleliste)} varer igjen å handle.")
