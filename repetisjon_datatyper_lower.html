Repetisjon av variabler og vilkår, introduksjon til datatyper, type(), konvertering og .lower().

Repetisjon: Variabler

En variabel er et navn som peker til en verdi, for eksempel tekst (str), heltall (int) eller desimaltall (float).

# Lage og bruke variabler
navn = "Anna"
alder = 15
hoyde = 1.65
print("Hei", navn)
print("Du er", alder, "år")
Merk: = brukes til å gi en variabel en verdi, mens == brukes til å sammenligne verdier.

Repetisjon: if / elif / else

# Grunnstruktur
if betingelse:
    print("Dette skjer hvis betingelsen er sann")
elif annen_betingelse:
    print("Dette skjer hvis den andre er sann")
else:
    print("Dette skjer ellers")

# Eksempel
poeng = 78
if poeng >= 90:
    print("Meget bra")
elif poeng >= 70:
    print("Bra jobbet")
else:
    print("Øv mer")

Hva er datatyper?

Python har flere innebygde datatyper; de mest brukte nå er str (tekst), int (heltall), float (desimaltall) og bool (True/False).

# Eksempler på datatyper
navn = "Anna"     # str (tekst)
alder = 15        # int (heltall)
hoyde = 1.65      # float (desimal)
er_elev = True    # bool (sann/usann)

type() viser hvilken type en verdi har, og hjelper med å forstå hvordan Python tolker variablene.

print(type(navn))     # <class 'str'>
print(type(alder))    # <class 'int'>
print(type(hoyde))    # <class 'float'>
print(type(er_elev))  # <class 'bool'>

input() gir tekst

input() returnerer alltid tekst (str), derfor må tall konverteres med int(...) før tallsammenligning.

# Uten konvertering: sammenligner tekst
a = input("Skriv et tall: ")  # "10"
b = input("Skriv et tall: ")  # "2"
print(a > b)  # True, fordi tekst sammenlignes alfabetisk

# Med konvertering: sammenligner tall
a = int(a)
b = int(b)
print(a > b)  # Riktig numerisk sammenligning
Obs: Hvis teksten ikke er et tall, vil int(...) feile; robust feilhåndtering kommer senere i kurset.

Konvertering mellom typer

# Til str (tekst)
tall = 42
tekst = str(tall)      # "42"
print(tekst)

# Til int (heltall)
tekst = "123"
tall = int(tekst)      # 123
print(tall)

# Til float (desimal)
tekst = "3.14"
des = float(tekst)     # 3.14
print(des)

.lower() for brukervennlig input

.lower() lager en ny versjon av teksten der alle bokstaver er små, nyttig for å sammenligne svar som “JA”, “Ja”, “jA” likt som “ja”.

# Uten .lower(): følsomt for store/små bokstaver
svar = input("Vil du fortsette? (ja/nei): ")
if svar == "ja":
    print("OK!")
elif svar == "nei":
    print("Stopper")
else:
    print("Skriv 'ja' eller 'nei'")

# Med .lower(): mer robust
svar = input("Vil du fortsette? (ja/nei): ").lower()
if svar == "ja":
    print("OK!")
elif svar == "nei":
    print("Stopper")
else:
    print("Skriv 'ja' eller 'nei'")
Tips: .lower() endrer ikke selve originalstrengen, men returnerer en ny, “småbokstav”-versjon.

Øvingsoppgaver

  1. Be om to tall, konverter med int(...), og skriv ut hvilket som er størst, eller at de er like.
  2. Be om alder som tekst, konverter til int(...), og skriv “barn” (<13), “ungdom” (13–17) eller “voksen” (18+).
  3. Be om “ja/nei” og bruk .lower() for å håndtere store/små bokstaver likt.
  4. Bruk type() på minst tre ulike variabler og forklar hva du ser.
