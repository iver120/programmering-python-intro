Python: Repetisjon + Datatyper

Repeterer variabler og vilkår, introduserer datatyper og .lower() metoden.

🔄 Repetisjon: Variabler

En variabel er som en etikett på en boks hvor vi lagrer verdier:

# Lage variabler
navn = "Anna"
alder = 15
høyde = 1.65

# Bruke variabler
print("Hei", navn)
print("Du er", alder, "år gammel")
Huskeregel: = er tilordning (gir en verdi), == er sammenligning (sjekker likhet).

🔄 Repetisjon: if / elif / else

# Grunnstruktur
if betingelse:
    print("Dette skjer hvis sant")
elif annen_betingelse:
    print("Dette skjer hvis 2. er sant")
else:
    print("Dette skjer ellers")

# Eksempel
poeng = 85
if poeng >= 90:
    print("Meget bra!")
elif poeng >= 70:
    print("Bra jobbet!")
else:
    print("Øv mer!")

🔄 Repetisjon: input() og strenger vs tall

Problemet vi oppdaget: input() gir alltid tekst, selv når vi skriver tall.

# Dette fungerer IKKE som forventet:
a = input("Tall: ")  # a blir "5" (tekst)
b = input("Tall: ")  # b blir "10" (tekst)
if a > b:            # "5" > "10" er True (alfabetisk!)
    print("a er størst")

# Løsning: Konverter til tall
a_tekst = int(input("Tall: ")) # int() Tvinger python til å lese input som heltall
b_tekst = int(input("Tall: ")) 
if a > b:         # Nå sammenligner vi tall
    print("a er størst")
🆕 Ny lærdom: Datatyper

Hva er datatyper?

Python skiller mellom ulike typer data. De viktigste er:

str (string/streng)
Tekst i anførselstegn
"Hei verden", "123"
int (integer/heltall)
Hele tall uten desimaler
42, -17, 0
float (desimaltall)
Tall med desimaler
3.14, -2.5
bool (boolean/sannhetsverdi)
Sant eller usant
True, False

Sjekke datatype med type()

# Sjekk hvilken type en variabel har
navn = "Anna"
alder = 15
høyde = 1.65
er_student = True

print(type(navn))      # <class 'str'>
print(type(alder))     # <class 'int'>
print(type(høyde))     # <class 'float'>
print(type(er_student)) # <class 'bool'>

# input() gir alltid str
svar = input("Skriv noe: ")
print(type(svar))      # <class 'str'> uansett hva du skriver

Konvertere mellom datatyper

# Til streng
tall = 42
tekst = str(tall)
print(tekst)  # "42" (nå er det tekst)

# Til heltall
tekst = "123"
tall = int(tekst)
print(tall)   # 123 (nå er det et tall)

# Til desimaltall
tekst = "3.14"
desimal = float(tekst)
print(desimal) # 3.14

# Til boolean (sjelden brukt)
print(bool("tekst"))  # True
print(bool(""))       # False
print(bool(0))        # False

Strengmetoder: .lower() for brukervennlighet

Strenger har innebygde metoder (funksjoner). Den første vi lærer er .lower():

# .lower() gjør alle bokstaver små
tekst = "HALLO VERDEN"
liten_tekst = tekst.lower()
print(liten_tekst)  # "hallo verden"

# Nyttig for input:
svar = input("Vil du fortsette? (ja/nei): ")
svar_liten = svar.lower()
if svar_liten == "ja":
    print("OK, vi fortsetter!")
elif svar_liten == "nei":
    print("Vi stopper her.")
Hvorfor .lower() er nyttig:
Brukere kan skrive "JA", "ja", "Ja" eller "jA" - alle blir til "ja" med .lower()

Sammenligning av strenger (alfabetisk)

Strenger sammenlignes i alfabetisk rekkefølge, men store bokstaver kommer før små [7]:

# Alfabetisk sammenligning
print("a" < "b")        # True
print("Anna" < "Bent")  # True
print("A" < "a")        # True (store bokstaver først!)

# Problem med stor/liten mix:
svar1 = "JA"
svar2 = "ja"
print(svar1 == svar2)   # False - de er ulike!

# Løsning med .lower():
print(svar1.lower() == svar2.lower())  # True - begge blir "ja"

Praktisk eksempel: Forbedret input

# Før (ikke robust):
svar = input("Fortsette? (ja/nei): ")
if svar == "ja":
    print("OK!")
# Fungerer bare hvis bruker skriver 
# nøyaktig "ja" med små bokstaver

# Etter (robust):
svar = input("Fortsette? (ja/nei): ").lower()
if svar == "ja":
    print("OK!")
elif svar == "nei":
    print("Stopper!")
else:
    print("Skriv 'ja' eller 'nei'")
# Fungerer med JA, Ja, jA, JA osv.

Oppsummering

Det vi har lært:
  • Python har ulike datatyper: str, int, float, bool
  • input() gir alltid str - konverter med int() for tall
  • .lower() gjør strenger til små bokstaver
  • Strenger sammenlignes alfabetisk
  • type() viser datatypen til en variabel
